Esta aplicación web es una prueba técnica desarrollada específicamente para demostrar habilidades en desarrollo full-stack moderno.

Objetivos de la Prueba

  • Implementar una arquitectura limpia y escalable
  • Demostrar conocimiento en tecnologías modernas
  • Aplicar mejores prácticas de desarrollo
  • Crear una interfaz de usuario moderna y responsiva

Frontend

  • Angular 17 (Standalone Components)
  • TypeScript 5.4
  • Tailwind CSS
  • RxJS (Reactive Programming)
  • Arquitectura Modular

Backend

  • NestJS (Node.js Framework)
  • MongoDB con Mongoose
  • JWT Authentication
  • Swagger/OpenAPI
  • TheCatAPI Integration
